api examples

Send a simple `POST` request with text, and the response is a text to speech (TTS) .wav file.

fetch example

    fetch("http://host-or-ip/submit", {
      "headers": {
        "content-type": "application/x-www-form-urlencoded",
      },
      "body": "text=With+this+you+can+generate+audio+from+text.",
      "method": "POST",
      "mode": "cors",
      "credentials": "omit"
    });

curl example

    curl 'http://host-or-ip/submit' \
    --data-raw 'text=With+this+you+can+generate+audio+from+text.' \
    --insecure
